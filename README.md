# Challenge_Telecom_Pt2
Telecom X – Parte 2: Predicción de Cancelación (Churn)  

# Objetivo
El objetivo sera desarrollar un pipeline analítico y modelos predictivos de Machine Learning capaces de anticipar qué clientes tienen la mayor probabilidad de cancelar sus servicios (churn). Esto con el fin de dotar a la empresa de una herramienta basada en datos que le permita tomar decisiones proactivas, optimizando sus estrategias de retención para evitar la pérdida económica antes de que ocurra.

# Estructura del proyecto
1. Preparacion de los datos
2. Correlacion y seleccion de variables
3. Modelado predictivo
4. Reporte Final

# Tecnologias y librerias utilizadas
Python 3.X. Pandas Matplotlib Numpy IPython.display  Seaborn Warnings sklearn.model_selection
